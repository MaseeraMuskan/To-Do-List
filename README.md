 To-Do-List:
A fully interactive and animated To-Do List web application built in a single HTML file (with embedded CSS &amp; JavaScript).  Features include deadline tracking, dynamic background transitions, task filtering (All / Completed / Missed), popup notifications with audio alerts, and local Storage data persistence.
An ultra-modern, fully interactive To-Do List web application built entirely in a single HTML file (with embedded CSS and JavaScript).
This project combines beautiful UI animations with powerful task management features like deadline tracking, notifications, and local storage support.

Features :
* Real-time clock display
* Add tasks with deadline (date & time)
* Mark tasks as completed
* Automatic detection of missed/overdue tasks
* Popup notifications for pending tasks
* Audio alert for reminders
* Animated background image slider
* Glassmorphism UI design with smooth animations
* Task statistics (Total / Completed / Missed)
* Data persistence using localStorage
* Responsive and interactive design

Built With:
HTML5
Embedded CSS3 (Animations & Effects)
Embedded JavaScript (DOM Manipulation, Notifications API, localStorage)
